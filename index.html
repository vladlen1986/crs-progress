How to Use
Tap a card to cycle statusEach tap moves the component: Planned → In Progress → Done → back to Planned. The left border colour shows current state.
Save File to persist changesClick "Save File" in the top bar after making changes. It downloads an updated copy of this HTML file with all statuses embedded. Replace your current file with the download and changes will be there next time you open it.
Filter by status or priorityUse the filter bar to narrow components across all modules. P1/P2/P3 filters stack with status filters.
Export CSVDownloads all modules, components, statuses, tags and descriptions as a spreadsheet-ready CSV file.
Add componentsClick "+ Add Component" in the top bar to add custom components to any module. They are saved locally alongside the built-in ones.
Navigate modulesUse the sidebar (desktop) or the hamburger menu (mobile) to jump to a specific module.
